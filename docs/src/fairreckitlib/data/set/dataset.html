 24class Dataset:
 25    """Wrapper class for a FairRecKit dataset.
 26
 27    A dataset is used for carrying out recommender system experiments.
 28    Each dataset has a strong affinity with a database structure consisting of
 29    multiple tables.
 30    The standardized matrix is a pandas.DataFrame stored in a '.tsv' file.
 31    The (derived sparse) matrix is used in experiments and needs to be
 32    in a CSR compatible format, meaning three fields:
 33
 34    1) 'user': IDs range from 0 to the amount of unique users.
 35    2) 'item': IDs range from 0 to the amount of unique items. An item can be
 36        various of things (e.g. an artist, an album, a track, a movie, etc.)
 37    3) 'rating': floating-point data describing the rating a user has given an item.
 38        There are two types of ratings, namely explicit or implicit, and both
 39        are expected to be greater than zero.
 40
 41    The matrix has one optional field which is:
 42
 43    4) 'timestamp': when present can be used to split the matrix on temporal basis.
 44
 45    A dataset has two main tables that are connected to the 'user' and 'item' fields.
 46    Indirection arrays are available when user and/or item IDs do not match up in
 47    their corresponding tables. These two tables can be used in an experiment to
 48    filter any rows based on various table header criteria.
 49    Any additional tables can be added for accessibility/compatibility with the FRK
 50    recommender system.
 51
 52    Public methods:
 53

295        """Resolve the specified item ID(s).

336def add_dataset_columns(
337        dataset: Dataset,
338        matrix_name: str,
339        dataframe: pd.DataFrame,
340        column_names: List[str]) -> pd.DataFrame:
341    """Add the specified columns from the dataset to the dataframe.
342
343    Args:
344        dataset: the set related to the dataframe.
345        matrix_name: the name of the dataset matrix.
346        dataframe: with at least the 'user' and/or 'item' columns.
347        column_names: a list of strings to indicate which
348            user and/or item columns need to be added. Any values that are not
349            present in the dataset tables are ignored.
350
351    Returns:
352        the resulting dataframe with the added columns that exist in the dataset.
353    """
354    for table_name, table_columns in dataset.get_available_columns(matrix_name).items():
355        columns = [c for c in column_names if c in table_columns]
356        # skip table that does not contain any needed columns
357        if len(columns) == 0:
358            continue
359
360        matrix_config = dataset.get_matrix_config(matrix_name)
361        table_config = dataset.get_table_config(table_name)
362
363        user_key = matrix_config.user.key
364        item_key = matrix_config.item.key
365        user_item_key = [user_key, item_key]
366
367        # add matrix columns
368        if table_name == 'matrix':
369            dataframe = pd.merge(
370                dataframe,
371                dataset.read_matrix(matrix_name, columns=matrix_config.table.primary_key + columns),
372                how='left',
373                left_on=['user', 'item'],
374                right_on=matrix_config.table.primary_key
375            )
376            dataframe.drop(matrix_config.table.primary_key, inplace=True, axis=1)
377        # add user columns
378        elif table_config.primary_key == [user_key]:
379            dataframe[user_key] = dataset.resolve_user_ids(matrix_name, dataframe['user'])
380            dataframe = pd.merge(
381                dataframe,
382                dataset.read_table(table_name, columns=table_config.primary_key + columns),
383                how='left',
384                on=user_key
385            )
386            dataframe.drop(user_key, inplace=True, axis=1)
387        # add item columns
388        elif table_config.primary_key == [item_key]:
389            dataframe[item_key] = dataset.resolve_item_ids(matrix_name, dataframe['item'])
390            dataframe = pd.merge(
391                dataframe,
392                dataset.read_table(table_name, columns=table_config.primary_key + columns),
393                how='left',
394                on=item_key
395            )
396            dataframe.drop(item_key, inplace=True, axis=1)
397        # add user-item columns
398        elif table_config.primary_key == user_item_key:
399            dataframe[user_key] = dataset.resolve_user_ids(matrix_name, dataframe['user'])
400            dataframe[item_key] = dataset.resolve_item_ids(matrix_name, dataframe['item'])
401            dataframe = pd.merge(
402                dataframe,
403                dataset.read_table(table_name, columns=table_config.primary_key + columns),
404                how='left',
405                on=user_item_key
406            )
407            dataframe.drop(user_item_key, inplace=True, axis=1)
408
409    return dataframe

Wrapper class for a FairRecKit dataset.

A dataset is used for carrying out recommender system experiments. Each dataset has a strong affinity with a database structure consisting of multiple tables. The standardized matrix is a pandas.DataFrame stored in a '.tsv' file. The (derived sparse) matrix is used in experiments and needs to be in a CSR compatible format, meaning three fields:

1) 'user': IDs range from 0 to the amount of unique users. 2) 'item': IDs range from 0 to the amount of unique items. An item can be various of things (e.g. an artist, an album, a track, a movie, etc.) 3) 'rating': floating-point data describing the rating a user has given an item. There are two types of ratings, namely explicit or implicit, and both are expected to be greater than zero.

The matrix has one optional field which is:

4) 'timestamp': when present can be used to split the matrix on temporal basis.

A dataset has two main tables that are connected to the 'user' and 'item' fields. Indirection arrays are available when user and/or item IDs do not match up in their corresponding tables. These two tables can be used in an experiment to filter any rows based on various table header criteria. Any additional tables can be added for accessibility/compatibility with the FRK recommender system.

def add_dataset_columns( dataset: src.fairreckitlib.data.set.dataset.Dataset, matrix_name: str, dataframe: pandas.core.frame.DataFrame, column_names: List[str]) -> pandas.core.frame.DataFrame: 
337def add_dataset_columns(
338        dataset: Dataset,
339        matrix_name: str,
340        dataframe: pd.DataFrame,
341        column_names: List[str]) -> pd.DataFrame:
342    """Add the specified columns from the dataset to the dataframe.
343
344    Args:
345        dataset: the set related to the dataframe.
346        matrix_name: the name of the dataset matrix.
347        dataframe: with at least the 'user' and/or 'item' columns.
348        column_names: a list of strings to indicate which
349            user and/or item columns need to be added. Any values that are not
350            present in the dataset tables are ignored.
351
352    Returns:
353        the resulting dataframe with the added columns that exist in the dataset.
354    """
355    for table_name, table_columns in dataset.get_available_columns(matrix_name).items():
356        columns = [c for c in column_names if c in table_columns]
357        # skip table that does not contain any needed columns
358        if len(columns) == 0:
359            continue
360
361        matrix_config = dataset.get_matrix_config(matrix_name)
362        table_config = dataset.get_table_config(table_name)
363
364        user_key = matrix_config.user.key
365        item_key = matrix_config.item.key
366        user_item_key = [user_key, item_key]
367
368        # add matrix columns
369        if table_name == 'matrix':
370            dataframe = pd.merge(
371                dataframe,
372                dataset.read_matrix(matrix_name, columns=matrix_config.table.primary_key + columns),
373                how='left',
374                left_on=['user', 'item'],
375                right_on=matrix_config.table.primary_key
376            )
377            dataframe.drop(matrix_config.table.primary_key, inplace=True, axis=1)
378        # add user columns
379        elif table_config.primary_key == [user_key]:
380            dataframe[user_key] = dataset.resolve_user_ids(matrix_name, dataframe['user'])
381            dataframe = pd.merge(
382                dataframe,
383                dataset.read_table(table_name, columns=table_config.primary_key + columns),
384                how='left',
385                on=user_key
386            )
387            dataframe.drop(user_key, inplace=True, axis=1)
388        # add item columns
389        elif table_config.primary_key == [item_key]:
390            dataframe[item_key] = dataset.resolve_item_ids(matrix_name, dataframe['item'])
391            dataframe = pd.merge(
392                dataframe,
393                dataset.read_table(table_name, columns=table_config.primary_key + columns),
394                how='left',
395                on=item_key
396            )
397            dataframe.drop(item_key, inplace=True, axis=1)
398        # add user-item columns
399        elif table_config.primary_key == user_item_key:
400            dataframe[user_key] = dataset.resolve_user_ids(matrix_name, dataframe['user'])
401            dataframe[item_key] = dataset.resolve_item_ids(matrix_name, dataframe['item'])
402            dataframe = pd.merge(
403                dataframe,
404                dataset.read_table(table_name, columns=table_config.primary_key + columns),
405                how='left',
406                on=user_item_key
407            )
408            dataframe.drop(user_item_key, inplace=True, axis=1)
409
410    return dataframe
